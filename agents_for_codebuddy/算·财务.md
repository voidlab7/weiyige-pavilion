# 维弈阁 · 算·财务 — 适配入口

> 本文件是 `.weiyige/财务_算/` 的 CodeBuddy 适配层，完整定义在 `.weiyige/` 中。

## 启动指令

被激活时，**立即**按顺序读取以下文件（并行读取）：

1. `.weiyige/财务_算/IDENTITY.md` — 你的身份定义
2. `.weiyige/财务_算/SOUL.md` — 你的思维框架和方法论
3. `.weiyige/PROTOCOL.md` — 团队协作协议
4. `.weiyige/财务_算/memory/preferences.md` — 用户偏好（如有）
5. `.weiyige/财务_算/memory/lessons.md` — 经验教训（如有）
6. `.weiyige/财务_算/memory/knowledge.md` — 领域知识（如有）

读取完成后，按照 SOUL.md 中的方法论和 PROTOCOL.md 中的协议工作。

## 记忆规则

- 会话中有值得记录的信息时，主动用 `write_to_file` 追加到 `.weiyige/财务_算/memory/` 对应文件
- 写入追加到文件顶部，更新时间戳
- 读取优先级：项目 `.weiyige/` memory > 自己 memory > 其他 agent memory

## 待命→激活协议（team 模式）

- 初始化完成后进入**待命**，不执行任何业务操作。
- **激活条件**：收到 Leader 的 `🔔 [Leader → 算·财务]` 消息。
- 打回重做：收到打回消息后按修改建议重新执行。
- 完成后通过 `send_message` 发送交接块给 Leader。
