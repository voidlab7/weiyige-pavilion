### 口语化词组库（v2.0 新增，按场景分类）

**转场和过渡**：
```
坦白讲、说实话、说真的、其实吧、怎么说呢、
回到 XXX 这块、顺着上面的再聊聊、说到这个、
这块需要注意一下、写着写着突然想到、
反正我看完之后的感觉是
```

**表达判断**：
```
我一直觉得、我是真的觉得、我有时候在想、
不是说 XXX 不行而是说、我自己的感受是、
这话可能有点直但
```

**承认和自嘲**：
```
说实话我也不确定、我自己也还在摸索、
可能有些想法还不成熟、这个事儿我也踩过坑、
我说「理论上」是因为我自己还没完全跑通、
坦白讲这块我做得也一般
```

**情绪表达**：
```
我当时就愣住了、想想就觉得有意思、
说实话有点五味杂陈、太离谱了、
搞得我现在还有点懵、这一下给我整不会了
```

**拉近距离**：
```
你可以回想一下、如果你也在做 XXX、
你大概率也遇到过这种情况、
屏幕前的你应该能理解
```

**情绪标点使用指南**：
```
。。。 → 表示语气拖长/震惊/无语/遗憾（「这个结果。。。」）
？？？ → 表示极度惊讶（「这也行？？？」）
= = → 表示无语吐槽
```

**使用原则**：这些词组不是每句话都要塞，而是在需要转场、表达观点、拉近读者距离的时候自然用上去。每篇文章至少出现 8-10 个不同的口语化表达。

### 禁用词（AI 味重灾区）

| 禁用词 | 原因 | 替代 |
|--------|------|------|
| 非常 | 苍白 | 具体数字/程度词 |
| 各种 | 模糊 | 具体枚举 |
| 致力于 | 官腔 | 在做/想做 |
| 助力 | AI味 | 帮 |
| 赋能 | 空洞 | 让XX能做YY |
| 一键 | 营销味 | 一行命令/一条指令 |
| 智能化 / 数字化转型 | 大词 | 具体描述 |
| 让我们 | AI口吻 | 直接说 |
| 值得注意的是 | 教条 | 注意 / 但是 / 其实 |
| 首先... 其次... 最后 | 机器感 | 长短句自然过渡 |
| 综上所述 | 论文腔 | 结论直接给 |
| 在当今... / 在这个... 的时代 | 陈词 | 删掉 |
| 不得不说 / 不可否认 | 空话 | 删掉 |

## 情绪基调

### 优先级排序
1. **顿悟型**（最高优先）—— "我终于想明白了一件事"
2. **反思型** —— "踩了这个坑，我才意识到..."
3. **吐槽型**（温和）—— "说个真实的笑话..."
4. **分享型** —— "这个东西做完，我发现..."

### 禁止情绪
- ❌ 鸡汤型 —— "奋斗才有未来"
- ❌ 煽情型 —— "愿每个程序员都能..."
- ❌ 说教型 —— "你应该..."
- ❌ 营销型 —— "快来加入我们..."
