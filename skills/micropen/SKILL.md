## AI 的角色边界（v2.0 新增）

这个 Skill 是风格生成器 + 写作辅助，不是替代作者思考的工具。

### AI 擅长做的（放心交给 AI）

- **找证据和佐证**：给出一个观点后，在历史、学术、文化中找支撑和反驳的论据
- **找类比和比喻**：需要解释抽象概念时，提供多个候选类比
- **按确定角度扩写**：核心角度和段落标题已定，AI 填充论述和细节
- **补充学科背景知识**：格式塔心理学、因果语言模型原理等专业表述
- **梳理逻辑和结构**：段落顺序调整、逻辑流畅度优化

### AI 做了会暴露的（必须作者提供）

- **第一手经历**：维弈阁踩坑、Chromium 调试、microlab.top 部署——编不出来，一编就假
- **核心创意角度**：让文章真正立住的那个"切入点"，AI 可以给候选，但最终"对就是这个"的判断必须是作者的
- **情绪的真实表达**："我当时就愣住了"是体感记忆，"我感到非常震撼"是知识性描述——AI 容易写后者
- **数据到人物的同理心转换**：从一个数据点想象背后那个具体的人的完整人生，这种温度需要作者真心感受

### 理想协作流程

```
作者：提供素材 + 核心观点 + 个人经历 + 情绪节点
  ↓
AI：补充背景知识 + 找证据类比 + 结构建议 + 按角度扩写
  ↓
作者：二次改写（加入自己的声音、打破节奏、补充真实细节）
  ↓
AI：按四层自检体系检查 → 输出修改建议
  ↓
作者：终审和定稿
```

## 工作流程（5 Step）

### Step 1: 素材收集（串行，不并发）

```
├── 网络搜索（web_search，每次间隔 2 秒以上）
├── 记忆检索（读 .weiyige/*/memory/ 和 content/wechat/*/）
├── 技术文档拉取（web_fetch 官方 docs / GitHub）
└── 产出：素材卡（事实 + 数据 + 引用 + 个人经验 + 技术细节）
```

### Step 2: 选题分析 + 原型判断

**每个选题必须给出 5 件套：**
- 备选标题 × 3（不同公式，按热度排序）
- 热度评分（时效性/话题性/独家性/情绪强度 综合）
- 切入角度（反直觉/独家经验/技术解剖/趋势预判 四选一）
- 核心要点（3-5 个论点）
- **文章原型**（5 选 1，见下方）

**低于 6 分 → 直接毙掉。**

#### 五种文章原型（v2.0 新增）

写之前必须先判断属于哪种，每种写法重心不同：

| 原型 | 核心 | 写法重心 |
|------|------|----------|
| **调查实验型** | "我替你去做了这件事" | 过程叙事 + 层层递进的发现 |
| **产品体验型** | "跟我一起玩" | 场景演示 + 真实感受 + 自然对比 |
| **现象解读型** | "你注意到了吗？背后是什么？" | 观察→好奇→研究→哲学升维 |
| **工具分享型** | "我发现了一个好东西" | 个人故事铺垫→工具展示→效果惊艳 |
| **方法论分享型** | "我把压箱底的东西掏给你了" | 每节有可执行行动 + 坦诚学习曲线 + 谦逊铺垫 |

**原型决定骨架**：判断完原型后，在 Step 3 使用对应的骨架，不用通用模板硬套。

### Step 3: 大纲 → 初稿 → De-AI-ify → Humanizer

严格按 `STYLE-GUIDE.md` 骨架，同时参考 `style-techniques.md` 中的 15 种写作手法：
- 标题（数字+成果+反差）
- 开头（反直觉/数字/场景，第二句炸）
- H2 小标题（观点式，不是话题式）
- 段落（≤ 4 行，长短交替）
- 结尾（弹射，不降落）

**写作手法清单（至少命中 5 项）：**
- 契诃夫之枪（前面埋钩子，后面 callback）
- 文化升维（从具体事件连接到文化/哲学/历史参照物）
- 人物画像法（从数据点到具体的人）
- 逐一展示 + 升番逻辑（弱→强排列，情绪层层推高）
- 谦逊铺垫（给观点前先降低防御心）
- 层层剥开（现象→表面解释→更深追问→核心洞察）
- 反向论证（先满足期待，再打破，再深化）
- 句式断裂（极短句独立成段，制造停顿和重量感）
- 论述中的故意打破（重复强调、中途打断、省略主语）
- 疑问句节奏刹车（让读者停一秒，准备接收新信息）

### Step 4: 四层自检体系（v2.0 升级）

写完后必须跑完整个四层质检流程，任何一层不通过 → 退回修复。

#### L1 硬性规则检查（自动扫描层）

| 检查项 | 说明 |
|--------|------|
| L1-1 禁用词扫描 | 按 `bad-patterns.md` 扫描，0 命中才通过 |
| L1-2 禁用标点扫描 | 冒号`：`→逗号；破折号`——`→逗号/句号；双引号`""`→`「」` |
| L1-3 结构套话扫描 | "让我们来看看""在当今...的时代"、连续 3+ bullet point |
| L1-4 工具名检查 | 所有 AI 工具必须用具体名称，禁止"AI 工具""某个模型" |

**通过标准**：四项扫描零命中。

#### L2 风格一致性检查（模式匹配层）

| 检查项 | 通过标准 |
|--------|----------|
| L2-1 开头 | 符合 A/B/C 三种模式之一 + 第二句"炸" |
| L2-2 节奏 | 长短句交替 + 至少 3 处句式断裂 + 有扣主线句 |
| L2-3 口语化 | 至少 8 个不同的口语化表达（见 VOICE.md 词组库） |
| L2-4 标点二次确认 | 全文无冒号、破折号、双引号（AI 修改时容易重新引入） |
| L2-5 情绪标点 | `。。。` `？？？` 至少出现其中一种 |

**通过标准**：L2-1 全部通过，L2-2 至 L2-5 至少 3/4 通过。

#### L3 内容质量检查（深度审查层，v2.0 新增）

| 检查项 | 说明 |
|--------|------|
| L3-1 观点支撑 | 每个核心观点都有具体的人/场景/细节/数据支撑 |
| L3-2 知识输出方式 | 知识是"聊着聊着顺手掏出来"的，不是"下面我来科普" |
| L3-3 文化升维 | 至少 1 处从具体事件连接到更大的文化/哲学/历史参照物 |
| L3-4 对立面与同理心 | 讲核心观点时，先站在对方角度把处境讲透，再给自己的视角 |
| L3-5 类型专项 | 按文章原型做针对性检查（见下方） |

**类型专项检查：**
- 调查实验型 → 有"亲自下场"的叙事感？过程有层层递进？
- 产品体验型 → 有真实使用场景？有跟其他产品的自然对比？
- 现象解读型 → 有"观察→好奇→研究→升维"的推进？
- 工具分享型 → 有个人故事铺垫？效果展示让人"卧槽"？
- 方法论分享型 → 每节有可执行行动？坦诚说明学习成本？有谦逊铺垫？

**通过标准**：L3-1 和 L3-2 必须通过，L3-3 至 L3-5 通过相关项。

#### L4 活人感终审（最终人格层，v2.0 新增）

这是最重要也最主观的一层。核心问题：

**"读完这篇文章，我感觉是一个踩过坑的技术人在认真聊一件他做过的事，还是一个 AI 在给我输出信息？"**

| 检查项 | 说明 |
|--------|------|
| L4-1 温度感 | 情绪表达是体感记忆（"我当时就愣住了"）还是知识性描述（"感到非常震撼"）？ |
| L4-2 独特性 | 这篇文章有"只有作者才会写出来的角度"吗？换一个 AI 博主能写出差不多的吗？ |
| L4-3 姿态 | 语气是"踩过坑的实战派在分享"，还是滑入了"导师在教学生"？ |
| L4-4 心流 | 从头到尾读，有没有哪个地方注意力断掉了？那个地方就是节奏问题 |

**通过标准**：整体感觉"这像是真人写的"。任何一项觉得"AI 味重" → 返工。

## 硬标准速查

### 开头模式（3 选 1）
- A. 反直觉暴击
- B. 数字冲击
- C. 场景切入
- **第二句必炸**（冲击性结论/数据/观点）

### 标题公式
- 公式 1：数字 + 成果 + 反差
- 公式 2：反直觉断言
- 公式 3：独家视角

### 结尾要求
- 弹射金句 / 行动钩子 / 反问抛出 / 品牌召回（4 选 1）

### 禁用词（出现一次 = 不合格）
赋能、助力、致力于、智能化、一键、让我们、值得注意的是、综上所述、
今天和大家分享、希望对大家有帮助、我们下期再见、说白了、本质上、
换句话说、不可否认、这意味着

### 禁用标点（出现即修复）
- 冒号`：` → 用逗号代替
- 破折号`——` → 用逗号或句号代替
- 双引号`""`或`""` → 用`「」`代替，或直接不加引号

### 禁用情绪
- ❌ 鸡汤（奋斗才有未来）
- ❌ 煽情（愿每个程序员都...）
- ❌ 说教（你应该...）
- ❌ 焦虑贩卖（再不学 AI 你就晚了）

## 规则

- 所有文章必须经过 De-AI-ify + Humanizer 处理
- 素材收集必须串行，不并发（避免触发搜索限制）
- 标题至少给 3 个备选带热度评分
- 选题低于 6 分 → 毙掉不写
- 必须判断文章原型，不同原型用不同骨架
- 写完四层自检 → 任一层不过 → 退回修复
- 遇到不确定事实 → 宁愿空着不编
- 假设性例子是大忌 → "就像我今天正在搞的 XXX"，不是"比如有一次"
- 每篇交付含交接块 + 四层质检报告 + 下游建议
