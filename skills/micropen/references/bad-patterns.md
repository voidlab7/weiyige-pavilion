## 一、绝对禁用词（出现一次 = 不合格）

### 大词 / 官腔
- 赋能
- 助力
- 致力于
- 智能化
- 数字化转型
- 生态闭环
- 链路打通（除非作为核心术语使用）
- 降本增效
- 多元化
- 全方位
- 战略性
- 重塑 / 重构行业
- 开创先河

### 空洞修饰词
- 非常
- 各种
- 一系列
- 无数
- 众多
- 大量
- 颇为
- 相当

### AI 口吻
- 让我们
- 值得注意的是
- 首先... 其次... 最后
- 综上所述
- 在当今 / 在这个时代
- 不得不说
- 不可否认
- 众所周知
- 显而易见
- 毋庸置疑

### 高频踩雷词（v2.0 新增，AI 特别爱用）
- "说白了" ← 一出现立刻暴露 AI，换成「坦白讲」「其实就是」
- "意味着什么？" / "这意味着" ← AI 标志性句式，换成「那结果会怎样呢」「所以呢」
- "本质上" ← 太学术，换成「说到底」「其实」
- "换句话说" ← 太书面，换成「你想想看」「也就是说」
- "不难发现" ← 直接删掉，直接说

### 营销腔
- 震惊
- 重磅
- 必看
- 干货
- 爆款（除非在讲内容领域）
- 一键
- 稳了
- 绝了
- 杀疯了
- 直接封神

## 一点五、标点禁令（v2.0 新增）

### 绝对禁用标点（出现即修复）

| 禁用标点 | 替代方式 |
|----------|----------|
| 冒号 `：` | 用逗号 `，` 代替 |
| 破折号 `——` | 用逗号或句号代替 |
| 双引号 `""` 或 `""` | 用 `「」` 代替，或直接不加引号 |

### 情绪标点（鼓励使用）

| 标点 | 用法 | 示例 |
|------|------|------|
| `。。。` | 语气拖长/震惊/无语/遗憾 | 「这个结果。。。」 |
| `？？？` | 极度惊讶 | 「这也行？？？」 |
| `= =` | 无语吐槽 | 「圈你妹 = =」 |

**扫描命令**：
```bash
grep -n "：\|——\|"\|"\|"\|"" draft.md
```

## 二、禁用句式

### ❌ 铺垫式开头
```
❌ "今天和大家分享一下..."
❌ "最近我一直在思考一个问题..."
❌ "相信很多朋友都遇到过..."
❌ "说起 XXX，大家的第一印象..."
❌ "在开始正文之前..."
```

### ❌ AI 典型过渡句
```
❌ "让我们来看看..."
❌ "接下来让我们分析一下..."
❌ "综上所述..."
❌ "总的来说..."
❌ "总而言之..."
```

### ❌ 降落式结尾
```
❌ "希望对大家有帮助"
❌ "欢迎在评论区交流"
❌ "我们下期再见"
❌ "一起加油"
❌ "愿我们都..."
```

### ❌ 过度对仗
```
❌ "既...又..."（连续 3 次以上）
❌ "不仅...而且..."（连续 3 次以上）
❌ "一方面...另一方面..."
```

## 三、危险情绪基调

### ❌ 鸡汤型
```
"奋斗才有未来"
"时代在抛弃你连招呼都不会打"
"每一个程序员都值得被看见"
```

### ❌ 煽情型
```
"愿每一个程序员都能..."
"致所有在路上的人..."
"向所有坚持的你致敬"
```

### ❌ 说教型
```
"你应该..."
"我们必须..."
"作为程序员，你有义务..."
```

### ❌ 焦虑贩卖型
```
"再不学 AI 你就晚了"
"这波红利你抓住了吗"
"程序员即将被淘汰"
```

## 四、结构 AI 味特征

### ❌ 过度工整
```
❌ 每段开头都是"1. 2. 3."
❌ 每个小节都是"定义 → 原理 → 应用"
❌ 段落长度完全相同
```

### ❌ 过度完备
```
❌ 每个观点都要"先说优点，再说缺点"
❌ 每个工具都要"适用场景、优缺点、总结"
❌ 凡事都要三段论：背景、分析、结论
```

### ❌ 无个人立场
```
❌ 只列事实，不给观点
❌ "有人认为 A，也有人认为 B"但没有自己的看法
❌ 怕冒犯谁，所有观点都模棱两可
```

## 六、替换建议对照表

| AI 味 | 人味替代 |
|-------|---------|
| 非常有用 | 真香 / 确实好使 / 离不开了 |
| 各种问题 | 3 个具体问题：XXX、YYY、ZZZ |
| 致力于构建 | 在做 / 想做 / 折腾 |
| 助力开发者 | 让开发者能 XXX |
| 赋能业务 | 让业务能 XXX |
| 一键部署 | 一行命令部署 / 一条指令部署 |
| 智能化分析 | 自动分析 / 让 AI 分析 |
| 让我们看看 | 来看（直接删掉"让我们"） |
| 值得注意的是 | 注意 / 但是 / 不过 / 其实 |
| 首先... 其次... 最后 | 一、二、三 / 直接分点，不用虚词 |
| 综上所述 | 结论：XXX / 直接给结论 |
| 希望对大家有帮助 | 删掉 |
| 我们下期再见 | 删掉 |
