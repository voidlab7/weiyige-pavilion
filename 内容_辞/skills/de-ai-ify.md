# 去 AI 味儿技能

## 概述

**来源**：孟健老师推荐的实用 Skill

AI 生成的文本通常有明显的特征：
- 过于正式、客套
- 用词重复（"非常"、"很好"、"重要"）
- 缺乏个性和情感
- 结构过于工整

本技能通过以下方式去掉"AI 味儿"：
1. **替换常见 AI 用词**
2. **增加口语化表达**
3. **打破工整结构**
4. **加入个性化细节**

---

## 使用场景

### 场景 1：公众号文章

**原文（AI 生成）**：
```
今天我要和大家分享一个非常重要的话题。这个话题对我们来说非常有价值。
通过深入研究，我发现了一些非常有趣的现象。这些发现对我们的工作非常有帮助。
```

**去 AI 味儿后**：
```
今天聊个事儿。
我研究了一天，发现了点有意思的东西。说实话，挺有用的。
```

### 场景 2：邮件回复

**原文（AI 生成）**：
```
非常感谢您的来信。我们已经收到了您的反馈。
我们会认真对待您的建议，并尽快给您回复。
再次感谢您的支持！
```

**去 AI 味儿后**：
```
收到了，谢谢反馈。
我们看看，有结果了马上告诉你。
```

### 场景 3：文案创作

**原文（AI 生成）**：
```
我们的产品非常优秀，功能非常强大，性能非常稳定。
我们致力于为用户提供最优质的服务，满足用户的各种需求。
```

**去 AI 味儿后**：
```
我们产品挺好用的，功能也够。
就是想让你用得爽，别的没啥。
```

## 去 AI 味儿的 7 个技巧

### 1. 替换常见 AI 用词

| AI 常用词 | 替换为 |
|----------|--------|
| 非常 | 挺、蛮、特别、超、相当 |
| 重要 | 关键、要紧、得注意 |
| 优秀 | 不错、可以、还行、挺好 |
| 致力于 | 想要、希望、就是要 |
| 提供 | 给、有 |
| 各种 | 各类、好多、一堆 |

### 2. 增加口语化表达

- "说实话"、"老实说"、"实际上"
- "有点儿"、"挺"、"蛮"
- "行吧"、"可以"、"没问题"
- "咋说呢"、"怎么讲"、"这么说吧"

### 3. 打破工整结构

❌ **AI 味儿重**（每句都是主谓宾）：
```
我们完成了项目。我们总结了经验。我们分享了成果。
```

✅ **自然**（长短句结合）：
```
项目做完了。总结了点经验，分享一下。
```

### 4. 加入个性化细节

❌ **AI 味儿重**（抽象泛泛）：
```
这个方法非常有效，帮助我们解决了很多问题。
```

✅ **自然**（具体细节）：
```
这个方法挺管用的，至少帮我省了 3 个小时。
```

### 5. 减少"的"字

❌ **AI 味儿重**：
```
这是一个非常重要的问题，需要我们认真的思考。
```

✅ **自然**：
```
这问题挺要紧，得好好想想。
```

### 6. 使用短句和停顿

❌ **AI 味儿重**（长句子）：
```
通过深入研究和分析，我们发现了一些非常有价值的现象，这些发现对我们的工作具有重要的指导意义。
```

✅ **自然**（短句 + 停顿）：
```
研究了一天。
发现了点有意思的东西。
对工作挺有用。
```

### 7. 加入情感和态度

❌ **AI 味儿重**（中性客观）：
```
这个项目已经完成了，效果还不错。
```

✅ **自然**（有情感）：
```
项目做完了，效果还行吧，至少没翻车。
```

## 实战示例

### 示例 1：技术文章开头

**AI 生成**：
```
大家好，今天我要和大家分享一个非常重要的技术话题。
这个话题在我们的日常工作中非常常见，也非常重要。
通过今天的分享，希望能够帮助大家更好地理解这个技术。
```

**去 AI 味儿**：
```
今天聊个技术问题，工作中经常遇到。
说实话，挺实用的。
```

---

### 示例 2：产品介绍

**AI 生成**：
```
我们的产品采用了先进的技术架构，具有非常强大的功能。
产品性能非常稳定，用户体验非常好。
我们致力于为用户提供最优质的服务。
```

**去 AI 味儿**：
```
我们产品技术还行，功能够用。
性能稳定，用起来挺顺手的。
就是想让你用得爽。
```

---

### 示例 3：经验分享

**AI 生成**：
```
通过这次项目，我学到了很多非常宝贵的经验。
这些经验对我的职业发展非常有帮助。
我希望能够和大家分享这些经验，帮助大家少走弯路。
```

**去 AI 味儿**：
```
这次项目踩了不少坑，学到点东西。
写出来分享一下，希望你们别再踩了。
```

## 常见误区

### 误区 1：过度口语化

❌ **错误**：
```
哎呀，这个真的超级超级好用啊！！！我真的太爱了！！！
```

✅ **正确**：
```
这个挺好用的，我挺喜欢。
```

### 误区 2：完全去掉所有"非常"

有时候"非常"是需要的（表达强烈程度）：

✅ **可以保留**：
```
这个问题非常严重，必须立即解决。
```

### 误区 3：只改用词，不改结构

❌ **不够**：
```
这个方法挺有效的，帮助我们解决了很多问题，对我们的工作挺有帮助的。
```

✅ **更好**：
```
这个方法挺管用，至少帮我省了 3 个小时。
```

## 检查清单

去 AI 味儿后，检查：

- [ ] 是否还有"非常"、"各种"、"致力于"等 AI 常用词？
- [ ] 是否每句都是工整的主谓宾结构？
- [ ] 是否缺乏口语化表达和停顿？
- [ ] 是否过于抽象，缺乏具体细节？
- [ ] 是否中性客观，缺乏情感和态度？
- [ ] 是否"的"字太多？
- [ ] 是否句子太长，没有短句和停顿？

## 技术实现

### 自动替换规则

```javascript
const aiWords = {
  '非常': ['挺', '蛮', '特别', '超', '相当'],
  '重要': ['关键', '要紧', '得注意'],
  '优秀': ['不错', '可以', '还行', '挺好'],
  '致力于': ['想要', '希望', '就是要'],
  '提供': ['给', '有'],
  '各种': ['各类', '好多', '一堆']
};
```

### 结构优化规则

1. 长句拆成短句（超过 20 字拆分）
2. 减少"的"字（"的"字数 / 总字数 < 5%）
3. 加入停顿（每 2-3 句加一个段落）

## 限制

1. **不适合正式文档** — 合同、报告、学术论文不适合去 AI 味儿
2. **不适合所有场景** — 根据受众和场合调整
3. **需要人工审核** — 自动去 AI 味儿后仍需人工检查
