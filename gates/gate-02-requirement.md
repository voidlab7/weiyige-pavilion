## 必查项（逐条勾选）

- [ ] PRD 包含一句话描述 + 问题假设
- [ ] MVP 最小闭环已定义
- [ ] **验收标准已结构化**（Given/When/Then 格式，鉴可直接转为测试）

- [ ] 时间表已拆到天级别
- [ ] 产品三角已审查（砺/矩/算 三角确认）
- [ ] PRD 已写入 `ai-workspace/{task-id}/artifacts/02-requirement/PRD.md`
- [ ] **PRD 文件已存在且可读**（通过 `read_file` 验证路径，非声称）

## 退出裁定

- ✅ 全部勾选 → 进入设计，交绘+矩
- ❌ 有未勾选项 → 枢补充后重新提交
