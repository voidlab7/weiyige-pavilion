> 审查方: 矩 | 退出条件: 代码通过左移检查 + 矩审查

## 必查项（逐条勾选）

- [ ] **左移检查已通过**（Lint + 类型检查 + 快速测试）
- [ ] 左移检查是通过 `execute_command` 实际执行的（非声称）
- [ ] Lint: 0 errors
- [ ] 类型检查: 0 errors
- [ ] 已有单元测试全部通过
- [ ] 代码符合架构方案中定义的组件边界
- [ ] 无越权修改（只改方案要求改的文件）
- [ ] 已提交 git commit
- [ ] **上游产物文件验证通过**（PRD、架构审查报告、设计审查报告均可 `read_file` 读取）

## 退出裁定

- ❌ 有未勾选项 → 修复后重跑左移检查
